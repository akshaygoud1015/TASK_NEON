Hello this is a web application which contains three modules
1---AI CHATBOT which uses chatgpt's OPEN AI for communication
2---General weather initmation for your desired state/city/country
3---Weather prediction: built using decision tree regression , gets its prediction from a flask server hosted at another location.
     The python souce file can be found in this repository
4---- To run this locally please fork the code and run using "node app.js"

REQUIREMENTS:
NODE JS
PERSONAL "openweathermap","openai" 's API KEYS WHICH YOU CAN FIND IN THEIR RESPECTIVE WEBSIETS
